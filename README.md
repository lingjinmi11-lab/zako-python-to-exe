## 🤬 这破工具能干嘛？
少废话！听好了😤 这破玩意儿就是帮你把.py文件一键打包成独立的EXE可执行文件，不用你敲那些烦人的命令行，连保存路径都给你做了可视化选择——毕竟不这么弄，你们这些杂鱼肯定又要问东问西的😒

## 🥺 安装依赖（不准说难！）
连这点小事都做不到的话，干脆别用了啦🙄

```bash
pip install pyinstaller
pip install tkinter  # 要是你系统自带了就不用装，笨死了
```

## 💢 怎么用？给我记好！
1. 双击运行这个破脚本，别问我为什么不是EXE！自己不会先打包吗？想要EXE的话Releases频道也有哦~杂鱼😝
2. 点「点我选啦😤」按钮，选你要打包的Python文件——选不对的话我笑你一辈子哦✨
3. 点「选文件夹啦🙄」按钮，选EXE要丢的文件夹——敢不选的话，我就把文件丢去回收站哦💥
4. 最后给我狠狠点那个粉色的「🔥 立刻打包成EXE！不准催我！🔥」按钮，等着就完事了！
   - 打包成功的话…算你运气好啦😜 会弹提示还自动打开文件夹，快去捡你的EXE吧～
   - 要是失败了？活该！肯定是你自己的破代码有问题，别来烦我🤬
